# Hale磁场分类法知识库

## 概述

Hale磁场分类法是由George Ellery Hale于1908年提出的太阳黑子磁场分类系统，是太阳物理学中最基础和广泛使用的分类方法之一。该系统基于太阳黑子的磁场极性分布特征，将太阳活动区分为不同的磁类型。

## 磁场基础

### 磁图解读

在SDO/HMI磁图中：
- **白色区域**: 正磁极（Positive polarity, N极性）
- **黑色区域**: 负磁极（Negative polarity, S极性）
- **灰色区域**: 平静太阳，代表弱磁场或中性线附近

### Zeeman效应

磁场的观测基于Zeeman效应：
- 磁场强度与光谱线分裂程度成正比
- 圆偏振方向指示磁场极性

## 分类类型

### Alpha (α) - 单极性黑子

**定义**: 单极性黑子群，只包含一个极性的黑子或紧凑的单一极性区域。

**特征**:
- 图像上只呈现单一颜色（全部白色或全部黑色）
- 没有明显的双极结构
- 磁场分布简单、紧凑
- 通常面积较小

**示例**: 小型孤立黑子、宁静区黑子

**空间天气意义**: 通常与弱耀斑活动无关

---

### Beta (β) - 双极黑子

**定义**: 具有两个明显分离的极性区域，正负极分界线清晰。

**特征**:
- 明显的双极结构（一个白色区域 + 一个黑色区域）
- 正负极之间有清晰的隔离带
- 分界线大致呈直线或平滑曲线
- 两极分离距离明显

**子类型**:
- **Beta-Gamma (β-γ)**: 双极结构，但分界线不规则或扭曲
- **Beta-Delta (β-δ)**: 在双极结构基础上包含delta配置

**空间天气意义**: 可产生中等强度耀斑，取决于面积和磁场强度

---

### Beta-Gamma (β-γ) - 混合极性双极黑子

**定义**: 双极黑子群，但正负磁极的分界线不规则、不平滑，存在极性交错。

**特征**:
- 整体呈双极结构
- 但正负极区域边界不规则
- 存在极性交错区域（一个小区域的正极嵌入负极群中）
- 磁场复杂性增加

**判别要点**:
- 不能用一条平滑曲线将正负极分开
- 极性分布呈现"之"字形或扭曲形态
- 混合程度比简单双极更复杂

**空间天气意义**: 与M级及以上耀斑有较强关联

---

### Delta (δ) - Delta黑子

**定义**: 相反极性的本影（umbra）被挤压在同一个半影（penumbra）内的复杂配置。

**特征**:
- 同一个半影内包含两个或多个极性相反的本影
- 正负极紧密混合
- 通常形成"日面内的日面"结构
- 磁场剪应力极大

**判别要点**:
- 找出一个半影区域内同时存在黑色和白色本影
- 极性分离距离极小
- 结构紧凑而复杂

**空间天气意义**: 已知与X级强烈耀斑和日冕物质抛射(CME)高度相关

```
开始分析
    │
    ▼

    │
    ├── 是 ──→ Alpha (单极性)
    │
    └── 否
        │
        ▼
    正负极分开明显吗？

        │    │
        │    ▼

        │    │
        │    ├── 规则 ──→ Beta (标准双极)

        │    └── 不规则 ──→ Beta-Gamma

        └── 否（极性混合）
             │

             │

             │
             └── 否 ──→ Beta-Gamma
```

## 实践技巧

### 1. 观察距离
- 保持适当观察距离，先看整体形态
- 再聚焦局部细节

### 2. 极性判断
- 白色 = 正极（N）
- 黑色 = 负极（S）
- 灰色 = 平静/弱磁场

### 3. 分界线分析
- 想象用一条线将正负极分开
- 如果这条线必须是弯曲/不规则的 = Beta-Gamma
- 如果无法用平滑曲线分开 = 更复杂的类型

### 4. Delta识别
- 寻找"岛"状结构
- 一个小极性的"岛"被困在另一个极性群中
- 这种配置高度不稳定

## 常见误区

1. **将噪音误认为极性混合**: 确保观察的是真实的信号而非图像噪点
2. **忽略半影结构**: Delta配置需要关注半影级别的细节
3. **过于依赖单一图像**: 最好结合多时次观测判断演化趋势
