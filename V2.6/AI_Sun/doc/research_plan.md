# 基于多模态大语言模型代理的太阳活动区自动分析系统

## 研究方案

---

### 1. 项目概述

#### 1.1 项目名称
**SolarInsight** —— 基于多模态大语言模型的太阳活动区自动分析系统

#### 1.2 研究背景
太阳活动区的自动识别与分类是空间天气研究的重要课题。传统的太阳黑子分类依赖于人工目视判断，效率低且主观性强。随着多模态大语言模型的发展，利用AI进行自动图像分析成为可能。

#### 1.3 研究目标
1. 利用 Helioviewer 获取高质量的多波段太阳图像
2. 构建基于 Coze 平台的多模态 AI 智能体，实现太阳黑子磁场类型（Hale分类）的自动化研判
3. 验证 AI 分析的准确性，建立"磁场-辐射"的物理关联

#### 1.4 技术路线
```
数据获取 → 图像预处理 → AI视觉分析 → 磁场分类 → 结果验证
    ↓              ↓              ↓              ↓
Helioviewer    Python脚本     Coze智能体     评估工具
```

### 2. 数据获取策略

#### 2.1 数据源
- **Helioviewer.org**: NASA SDO 数据可视化工具
- **数据类型**: 
  - HMI 磁图 (Magnetogram)
  - AIA 极紫外图 (EUV 171/193)

#### 2.2 数据采集方法
1. **时间选择**: 利用 HEK Events Timeline 选择活跃期
2. **图像截取**: 使用 Select Area 功能获取高分辨率局部图
3. **命名规范**: `日期_活动区编号_波段.jpg` (如 `20240510_AR3664_HMI.jpg`)

#### 2.3 数据集构建
- 训练集: 100 张标注好的磁图
- 测试集: 20 张独立测试样本

### 3. Coze 智能体设计

#### 3.1 系统提示词设计

```markdown
# Role
你是一位专业的太阳物理观测助手，擅长通过视觉分析太阳动力学天文台(SDO)的图像。

# Skills
1. **磁图识别**: 识别HMI磁图中的黑白区域，白色代表正磁极，黑色代表负磁极
2. **分类推理**: 基于Hale分类法判断黑子类型

# Hale分类标准
- **Alpha (α)**: 单极性黑子，只有一个极性区域
- **Beta (β)**: 双极结构，正负极分开明显
- **Beta-Gamma (β-γ)**: 双极组，但正负极分界线不规则，互相交错
- **Delta (δ)**: 极端复杂，相反极性的本影被挤压在同一个半影内

# Workflow
1. **图像确认**: 确认是否为太阳图像
2. **视觉描述**: 详细描述黑子群的形态特征
3. **逻辑判断**: 分析黑白区域的混合程度和空间关系
4. **结论输出**: 给出最可能的Hale分类，并说明置信度

# Constraints
- 如果图片模糊，建议用户截取高清局部图
- 保持科学严谨的态度，明确标注分类的不确定性
```

#### 3.2 知识库配置
- **知识库名称**: `Solar_Physics_Wiki`
- **内容**: Hale磁场分类法标准定义、典型示例图
- **关联方式**: 启用RAG检索增强

#### 3.3 工作流设计
```
Start → LLM节点(视觉分析) → LLM节点(推理分类) → End
         ↓                        ↓
    图像描述                  结合知识库输出结果
```

### 5. 研究实施步骤

#### 第一阶段：数据准备（第1-2周）
1. 学习 Helioviewer 操作流程
2. 截取20张不同形态的黑子磁图
3. 收集对应的NOAA官方分类作为标签

#### 第二阶段：智能体构建（第3-4周）
1. 在Coze平台创建智能体
2. 配置系统提示词和知识库
3. 设计工作流节点

#### 第三阶段：测试与迭代（第5-6周）
1. 上传测试图片进行初步测试
2. 评估AI识别准确性
3. 优化提示词和知识库

#### 第四阶段：多模态验证（第7-8周）
1. 获取同一活动区的HMI磁图和AIA极紫外图
2. 验证AI建立"磁场-辐射"关联的能力
3. 生成研究报告

### 6. 评估指标

| 指标 | 计算方法 | 目标值 |
|------|----------|--------|
| 分类准确率 | 正确分类数/总样本数 | ≥80% |
| Alpha分类精度 | Alpha正确数/Alpha样本数 | ≥85% |
| Beta分类精度 | Beta正确数/Beta样本数 | ≥80% |
| 复杂类型精度 | (Beta-Gamma+Delta)正确数/复杂类型样本数 | ≥70% |
