# 太阳活动区自动分析系统 - 用户手册

## 概述

欢迎使用太阳活动区自动分析系统！本系统基于多模态大语言模型，能够自动分析太阳活动图像并生成专业的分析报告。

---

## 一、系统功能

### 1.1 核心功能
- **图像上传**: 支持上传太阳观测图像（JPG/PNG/TIFF格式）
- **自动分析**: 利用AI模型识别太阳黑子、耀斑、磁场特征等
- **报告生成**: 自动生成结构化的分析报告
- **风险评估**: 评估太阳活动的潜在风险等级
- **案例库**: 提供预设的太阳活动案例供学习参考

### 1.2 适用场景
- 太阳观测数据的快速分析
- 太阳活动的初步筛查
- 教学演示和科普教育
- 科研辅助工具

## 三、界面功能说明

### 3.1 主界面

| 区域 | 功能 | 说明 |
|------|------|------|
| 顶部导航 | 系统菜单 | 包含首页、分析、案例库、历史记录等 |
| 左侧面板 | 功能入口 | 快速访问各项功能 |
| 主内容区 | 操作区域 | 显示当前功能的操作界面 |
| 底部状态栏 | 系统状态 | 显示当前状态、时间等信息 |

### 3.2 图像上传

**操作步骤**:
1. 点击「上传图像」按钮
2. 选择本地太阳图像文件
3. 填写图像元信息（可选）
   - 数据源（如SDO/HMI）
   - 波长（如171A）
   - 观测时间
4. 点击「确认上传」

**支持的图像格式**:
- JPG/JPEG
- PNG
- TIFF
- FITS（建议转换为上述格式）

**图像要求**:
- 分辨率：建议≥1024x1024像素
- 文件大小：建议≤10MB
- 内容：太阳盘面清晰可见

### 3.3 图像分析

**分析选项**:
| 选项 | 说明 | 默认值 |
|------|------|--------|
| 分析类型 | basic（基础）/ full（完整） | full |
| 风险评估 | 是否包含风险评估 | 是 |
| 摘要生成 | 是否生成分析摘要 | 是 |
| 详细模式 | 是否输出详细分析结果 | 否 |

**分析内容**:
1. **黑子区域分析**: 识别黑子群编号、位置、大小、磁类型
2. **亮区特征分析**: 识别耀斑、日珥、谱斑等
3. **磁场特征分析**: 磁场强度、极性分布、复杂性
4. **风险评估**: 活动风险等级和预测

**案例类型**:
| 类型 | 说明 |

## 四、API使用说明

### 4.1 接口调用示例

**使用Python调用**:

```python
import requests

# 上传图像
with open('sdo_image.jpg', 'rb') as f:
    response = requests.post(
        'http://localhost:8000/api/v1/images',
        files={'image': f},
        data={'source': 'SDO/HMI', 'wavelength': '171A'}
    )
image_id = response.json()['data']['id']

# 执行分析
response = requests.post(
    'http://localhost:8000/api/v1/analyze',
    json={
        'image_id': image_id,
        'analysis_type': 'full',
        'options': {
            'include_risk_assessment': True,
            'include_summary': True
        }
    }
)
task_id = response.json()['data']['task_id']

# 获取分析结果
import time
while True:
    response = requests.get(f'http://localhost:8000/api/v1/analyze/{task_id}')
    status = response.json()['data']['status']
    if status == 'completed':
        report_id = response.json()['data']['report_id']
        break
    elif status == 'failed':
        raise Exception('分析失败')
    time.sleep(2)

# 获取报告
response = requests.get(f'http://localhost:8000/api/v1/reports/{report_id}')
report = response.json()['data']
print(report['summary'])
```

**使用curl调用**:

```bash
# 上传图像
curl -X POST http://localhost:8000/api/v1/images \
  -F "image=@sdo_image.jpg" \
  -F "source=SDO/HMI" \
  -F "wavelength=171A"

# 执行分析
curl -X POST http://localhost:8000/api/v1/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "image_id": "img-20260320-120000",
    "analysis_type": "full"
  }'

# 获取报告
curl http://localhost:8000/api/v1/reports/rpt-20260320-120500
```

## 五、常见问题

### 5.1 图像上传失败

**可能原因**:
- 文件格式不支持
- 文件大小超过限制
- 网络连接问题

**解决方法**:
- 确保文件为JPG/PNG/TIFF格式
- 压缩图像大小
- 检查网络连接

### 5.2 分析时间过长

**可能原因**:
- 图像分辨率过高
- 服务器负载较高
- 模型调用延迟

**解决方法**:
- 适当降低图像分辨率
- 等待系统空闲时重试
- 检查服务器状态

### 5.3 分析结果不准确

**可能原因**:
- 图像质量较差
- 太阳盘面不清晰
- 模型识别限制

**解决方法**:
- 上传高质量图像
- 确保太阳盘面居中且完整
- 参考案例库对比分析

### 5.4 如何获取太阳观测数据

**推荐数据源**:
- **SDO (Solar Dynamics Observatory)**: https://sdo.gsfc.nasa.gov/
- **SOHO (Solar and Heliospheric Observatory)**: https://soho.nascom.nasa.gov/
- **Hinode**: https://hinode.nao.ac.jp/
- **中国科学院国家天文台**: http://www.bao.ac.cn/

**数据获取方法**:
1. 访问上述网站
2. 选择所需波长和时间
3. 下载图像文件
4. 上传至本系统进行分析

## 六、注意事项

### 6.1 数据安全
- 上传的图像仅用于分析，不会用于其他用途
- 分析结果仅保存在服务器指定时间
- 建议敏感数据自行备份

### 6.2 结果解读
- 本系统提供的分析结果仅供参考
- 专业研究请结合其他数据综合判断
- 如有疑问请咨询专业太阳物理学家

### 6.3 系统限制
- 单次分析时间可能较长（10-60秒）
- 图像大小建议不超过10MB
- 每日分析次数可能有限制
